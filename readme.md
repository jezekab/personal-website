# Attempt at a ReactJS based them for Wordpress

## Implementation

* ReactJS
* RefluxJS
* WP-API (server plugin and node.js client)
* Gulp + Browserify

## Getting Up & Running

Install dependencies with npm.

```
npm install
```

Adding a dependency

```
npm install package-name --save
```

## Building

To bundle JS and compile LESS:
```
gulp build
```

For watch & browser-sync:
```
gulp watch
```
